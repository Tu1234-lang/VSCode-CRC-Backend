Cloud Resume Challenge - Azure:

1. Goal: Create a static website using Microsoft Azure, then manage and secure the domain with CloudFlare. 
  - Bonus: Utilize Cloud Security and Web App Security concepts to enhance learning and hand-on experience
2. Technologies learned: HTML, CSS, Azure Storage, Static Web, Azure Functions, HTTPS Triggers Functions, CloudFlare DNS, Javascript, Azure CosmosDB, Azure Table API, Python, Azure Python SDK, Github, Git
  - Bonus: CloudFlare (WAF, Certificates, TLS/SSL, HTTPS), Azure (Microsoft Defender for Cloud), GitHub (GPG - signed commit, CodeQL)
3. Why do the challenge: Gain hand-on experience on managing a personal website and cloud; IMPORTANT- Web Application Security and Cloud Security
